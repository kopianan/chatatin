Aplikasi ini digunakan untuk mengelola komunikasi, otomatisasi pesan, dan integrasi webhook pada akun Instagram Business.

1. Data yang Dikumpulkan

Kami dapat mengumpulkan data terbatas dari pengguna, termasuk:

  • Username Instagram
  • User ID
  • Isi pesan (Direct Message)
  • Metadata pesan untuk keperluan otomatisasi

2. Penggunaan Data

Data digunakan hanya untuk:

  • Menjawab dan mengelola pesan pengguna
  • Otomatisasi layanan
  • Integrasi sistem internal dan webhook
  • Peningkatan kualitas layanan

Kami tidak menjual, menyewakan, atau menggunakan data pengguna untuk tujuan pemasaran eksternal.

3. Penyimpanan dan Keamanan Data

Data disimpan secara aman menggunakan sistem internal dan/atau layanan pihak ketiga yang memiliki standar keamanan yang layak. Kami mengambil langkah teknis dan organisasi yang wajar untuk melindungi data pengguna.

4. Pembagian Data

Kami tidak membagikan data pengguna kepada pihak ketiga, kecuali jika diperlukan untuk menjalankan layanan atau diwajibkan oleh hukum.

5. Penghapusan Data

Pengguna dapat meminta penghapusan data pribadi kapan saja dengan menghubungi kami melalui email di bawah ini.
